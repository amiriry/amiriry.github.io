What is inifity?

Its sgin is: ∞
But what does that mean?
You can look at it as a process, some kind of a model for an unending process.
If you want to define a process that never stops - define it with infinity.

Example1
Lets try to find the biggest number exists.
Obviously, we cannot think of one... No matter how hard we search for it, we can find a number that is bigger than him if we only do the simple task of adding 1...
So if we now we can add a number to whatever number we find.

So instead of finding the biggest number exist, lets define the process of finding it.
You encountered a number? take a number that is bigger than it.
And now define something for infinity.
BUT HOW CAN WE IF IT DOESN'T EXIST??!!!
As the process above, if we define something for inifinity, it means that no matter how far we go the definition still exist.
That's how, instead of defining what happens exactly at a place we don't know (infinity) we define a process that happens in "infinity"
For example something that happens:
1. When we get close to a number:
x → 1
x → 0
2. When we try to get infinity:
x → ∞
All of them are some kind of inifity.
If you are approaching a number, just substract the fixed number and you get approaching to zero.
Approaching to zero is just 1 devided by infinity in some way.
In the end of all of this, everything revolves around what inifinity represents.
